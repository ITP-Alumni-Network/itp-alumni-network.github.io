Using our Github Repos

Documents and media will be maintained in repositories on Github with the intention of letting anyone in the community fork and update as needed. Currently there are two public repositories for this Github group:
  • itp-alumni-network.github.io - This contains the HTML for this site.
  • TheFloor - This will contain any files, media, notes, and whatnot related to the ITP Alumni Network initiative.
These repos are fully open to the public. We urge you to fork them and add any changes/updates/files you'd like and submit a pull request to add them to the master branch.
